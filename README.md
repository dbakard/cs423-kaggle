# COSC 423 Final Project - [I'm Something of a Painter Myself](https://www.kaggle.com/competitions/gan-getting-started)

### Steps:
1. Create a Kaggle account and verify your phone number.
2. Go to this link https://www.kaggle.com/code/amyjang/monet-cyclegan-tutorial and click Copy & Edit notebook to create your own copy of the baseline model.
3. Update the GCS_PATH to GCS_PATH = '/kaggle/input/monet-gan-getting-started'
4. Click Settings -> Accelerator -> GPU T4 x2 (TPU might be better idk)
5. Run all cells and wait for it to finish.
